# 命苦换算器

一个让你认清现实的消费计算器——把每一分钱都换算成你生命的代价！

## 功能特点

### 🔢 核心换算
输入你想买的东西的价格，马上算出你需要卖命多久才能买得起：
- 按**天命**（天）计算
- 按**工时**（小时）计算
- 按**分钟**计算

### 🎵 沉浸式BGM
配有喜感又略带命苦的背景音乐，让你在剁手时更有氛围

### 📋 牛马欲望清单
把你想要的东西都记下来，看看你的欲望总共值多少年寿命

### 💌 发疯文学离职信
根据你的薪资水平，自动生成一封适合你的离职信

### 👑 职业生涯总估值
算算你这辈子能赚多少碗猪脚饭，多少个花西子

## 使用说明

1. 设置你的**税后月薪**和**月工作天数**
2. 输入你想买的东西的**价格**
3. 看看需要付出多少生命的代价
4. 点击「点击补刀」获取一句扎心名言

## 技术栈

- React 18
- Vite
- Tailwind CSS
- lucide-react 图标库
